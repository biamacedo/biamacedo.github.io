Projects

Some personal projects I have worked on.

Hacker News Reader

Shows news from Y Combinator website.
Android mobile app, published on Google Play.
Created using: Apache Cordova, Ionic Framework, Angular JS, Hacker News Firebase API (REST) and Search HN Algolia API (REST).

Pokedex

iOS mobile app that shows information about all 151 pokemons.
Created using: Pokéapi - The Pokémon RESTful API

Web Chat Application

Web chat application that uses a node.js server as a back-end, hosted at Heroku.
Created using: Bookstrap, jQuery, Socket.io, express.js and moment.js.

Todo Notes REST API

Node API that keeps track of your todo's, hosted at Heroku.
Created using a SQLite database and the following frameworks: express.js, body-parser, underscore.js and sequelize.

Tip Calculator

Calculates the tip for your bill and splits the amount between a number of people.
Android mobile app, published on Google Play.
Created using: Apache Cordova and JQuery Mobile.

Work Projects

Projects I participated/created at work.

Coverage Tool Back-end

Node JS Rest API that interacts with a db2 database and retrieves information about the area's coverage to the sales team, mostly used on the month's closing.
To be hosted on Bluemix.
Created using the following frameworks: express.js, body-parser, underscore.js and ibm_db to interact with db2.

My Spot App

Hybrid app (to be used on Android and iOS) with node API as back-end, that lets users reserve a workstation when they have to go to the office and check-in when they arrive.
App created using IBM MobileFirst Worklight and Framework7.
Back-end node API created using: express.js, body-parser and ibm_db to interact with db2.
